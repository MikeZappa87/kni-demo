# kni-demo

## Prerequisite

- [kind](https://github.com/kubernetes-sigs/kind)
- [Task](https://taskfile.dev/)
- [golang](https://go.dev/)


## How to Setup

```
$ git clone https://github.com/MikeZappa87/kni-demo
$ cd kni-demo
$ task 01-init
$ task 02-build
$ task 03-setup
```

## How to teardown

```
$ task cleanup
```
